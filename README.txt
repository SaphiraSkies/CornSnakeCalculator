===== CS 361 Corn Snake Calculator =====
Hannah Moon


Description:

This app is meant for corn snake breeders who are looking to find out what kind of morphs (color variations) they will get when breeding two snakes together.

It was built with Python for a Windows machine.

*******************************

To run:

1. Install dependencies:
	pip install Pillow

2. Account service must be running:
	https://github.com/PaulSVander/CS361-accounts-microservice

2. Compile:
	python3 gui.py
